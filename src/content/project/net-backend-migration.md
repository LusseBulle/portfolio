## Overview
Working alongside a student team for a corporate client, we modernized a critical piece of server infrastructure. The core objective was migrating an aging, legacy TypeScript-based solution into a more robust and scalable ecosystem.

### Architecture & Migration
We rebuilt the backend architecture from the ground up using C# and .NET. We integrated Entity Framework Core for reliable, structured database access and management. The modernized architecture was designed specifically for cloud deployment on Microsoft Azure, ensuring enterprise-grade scalability. 

### Team Collaboration
The migration was executed in agile sprints using Scrum methodologies. This allowed us to systematically decouple and transition legacy endpoints into the new .NET environment while maintaining strict alignment with the client's operational requirements.
