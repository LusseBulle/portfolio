## Overview
As part of an industry collaboration in a student-team, we developed a proof-of-concept Model Context Protocol (MCP) server designed to bridge the gap between AI agents and existing corporate DevOps workflows. 

### Technical Implementation
The server itself was built in Go. By leveraging the MCP standard, the application provides a standardized interface that allows AI agents to securely interact with and automate specific infrastructure tasks within a Kubernetes environment. 

### Process & Outcome
This project successfully demonstrated how modern AI tooling can be integrated into strict corporate infrastructures without compromising operational security or stability.
